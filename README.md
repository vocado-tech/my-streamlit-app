✈️ NoRegret Trip 사용 설명서

1. 개요

이 프로그램은 여행지 결정에 어려움을 겪는 사용자를 위해 제작된 AI 기반 여행지 추천 서비스입니다. 사용자가 입력한 기간과 취향을 분석하여 현실적으로 여행 가능한 도시 3곳을 지도, 일정, 예산과 함께 제안합니다.

2. 주요 기능
 A. 맞춤형 여행지 추천 로직

* 여행 기간 연동: 여행 기간에 따라 추천 가능한 지리적 범위를 자동으로 제한합니다.
  단기(1~3일): 이동 시간이 짧은 동북아시아 위주 추천
  장기(일주일 이상): 유럽, 미주, 대양주 등 전 세계 추천


* 여행 난이도 설정:
  쉬움: 인프라가 편하고 한국인이 선호하는 대중적인 여행지 추천
  모험가: 오사카, 다낭 등 흔한 여행지를 제외하고 이색적인 여행지 추천



 B. 상세 정보 제공

* 예산 산출: 항공, 숙박, 식비를 포함한 1인 기준 총 예상 비용 제공
* 일정 안내: 해당 도시에서 소화 가능한 현실적인 주요 동선 안내
* 안전 확인: 외교부 지정 여행 금지 국가는 추천 목록에서 자동 제외

 C. 편의 기능

* 위치 시각화: 추천 도시의 위도/경도를 기반으로 지도(Map)에 위치 표시
* 항공권 연결: [최저가 확인] 버튼 클릭 시 해당 도시행 스카이스캐너 검색 결과로 즉시 이동



3. 사용 방법 (User Guide)

 1. 초기 설정: 앱 실행 후 사이드바에 OpenAI API Key를 입력합니다.
 2. 조건 입력:
  * 여행 기간, 동행인, 선호하는 여행 스타일(휴양/관광 등)을 선택합니다.
  * 예산 수준(가성비/럭셔리)을 설정합니다.
  * 추가 요청 사항(ex 이 나라 빼고 추천해주세요, 더운 나라는 싫어요)을 입력합니다.


 3. 여행 난이도 선택:
  * 편안한 여행을 원하면 '쉬움', 여행 경험이 많고 색다른 곳을 원하면 '모험가'를 선택합니다.


 4. 결과 확인:
  * `추천받기` 버튼을 클릭하면 3개의 탭으로 추천 여행지가 표시됩니다.
  * 지도와 상세 내용을 확인 후, 하단 버튼을 통해 항공권을 검색합니다.



---

 4. 개발 환경 (Tech Stack)

* Language: Python 3.9
* Framework: Streamlit
* AI Engine: OpenAI GPT-4o-mini
* Libraries: Pandas(지도 데이터 처리), Requests(API 통신)

 5. 설치 및 실행 (Installation)

로컬 환경에서 코드를 실행하려면 아래 명령어를 순서대로 입력하십시오.

```bash
# 1. 저장소 복제
git clone https://github.com/your-username/my-streamlit-app.git

# 2. 필수 패키지 설치
pip install -r requirements.txt

# 3. 애플리케이션 실행
streamlit run app.py

```

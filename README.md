# NoRegret Trip

AI 기반 여행 의사결정 지원 웹 애플리케이션

---

## 1. 프로젝트 개요

NoRegret Trip은 여행지 선택 과정에서 발생하는 의사결정 부담을 줄이기 위해 설계된 AI 기반 여행 설계 지원 서비스이다.

사용자의 여행 기간, 예산, 동행 여부, 선호 스타일 등의 조건을 종합적으로 분석하여 현실적으로 실행 가능한 여행지를 제안하며, 단순 추천을 넘어 다음과 같은 요소를 함께 제공한다:

* 일정 초안
* 계절 및 날씨 리스크 분석
* 예산 추정
* 대표 관광지 및 음식 정보
* 도시 생활 환경 평가
* 입국 조건 안내
* 후속 재추천 챗봇 기능

본 프로젝트는 여행 정보 제공을 넘어, 사용자의 “결정 과정”을 지원하는 것을 목표로 한다.

---

## 2. 주요 기능

### 1) 맞춤형 여행지 추천

* 여행 기간 기반 지리적 범위 제한
* 여행 난이도(쉬움 / 모험가) 기반 필터링
* 예산 수준 반영
* 현실적으로 가능한 도시 3곳 제안

---

### 2) AI 기반 일정 수립 및 재추천 챗봇

* GPT 기반 도시별 일정 초안 생성
* 사용자가 "마음에 들지 않음" 요청 시 대체 여행지 제안
* 일정 수정, 관광지 추천 등 후속 대화 지원
* 조건 반영형 대화형 인터페이스 제공

---

### 3) 계절 및 기후 리스크 분석

* 월별 평균 기온 및 강수량 기반 분석
* 우기 및 태풍 시즌 자동 감지
* 여행 시 감수해야 할 리스크 안내
* 평균 기온 계산을 통한 시기 적합성 판단

---

### 4) 대표 이미지 및 관광지 정보 제공

* Unsplash 기반 도시 대표 이미지 자동 로드
* Wikipedia 및 보조 검색 기반 이미지 확보
* 주요 랜드마크 이미지 다중 표시
* 관광지 정보 요약 제공

---

### 5) 대표 음식 및 로컬 푸드 추천

* 국가 기반 대표 음식 자동 매핑
* TheMealDB API 활용 레시피 정보 제공
* 음식 이미지 및 출처 링크 표시
* 국가별 로컬 푸드 추천 기능

---

### 6) 도시 생활 환경 분석 (Teleport API 기반)

* Safety (안전도)
* Cost of Living (물가 수준)
* Leisure & Culture (여가/문화)
* Environmental Quality (환경)

Teleport 점수를 기반으로 도시 장점과 주의점을 자동 생성하여 사용자에게 설명한다.

---

### 7) 항공권 검색 연동

* 입력 날짜 기반 스카이스캐너 검색 URL 자동 생성
* 공항 코드 기반 목적지 연결

---

### 8) 입국 및 비자 조건 안내

* 주요 국가별 한국 여권 기준 입국 조건 정리
* 무비자/전자비자/여권 유효기간 조건 안내

## 5. 실행 방법

### 1) 저장소 클론

```bash
git clone https://github.com/vocado-tech/my-streamlit-app.git
cd my-streamlit-app
```

### 2) 패키지 설치

```bash
pip install -r requirements.txt
```

### 3) API 키 설정

`.streamlit/secrets.toml` 파일 생성 후 다음 항목 추가:

```
OPENAI_API_KEY = "your_openai_key"
OPENWEATHER_API_KEY = "your_openweather_key"
OPENTRIPMAP_API_KEY = "your_opentripmap_key"
```

Teleport API 및 TheMealDB는 기본적으로 인증 없이 사용 가능하다.

### 4) 앱 실행

```bash
streamlit run app.py
```

## 8. 프로젝트 의의

NoRegret Trip은 기존 여행 플랫폼과 달리 단순 정보 나열이 아닌, 사용자의 의사결정을 지원하는 통합형 여행 설계 도구를 목표로 한다.

정성적 AI 분석과 정량 데이터(API)를 결합하여, 여행 계획 과정의 불확실성과 후회 가능성을 줄이는 것을 핵심 가치로 한다.
